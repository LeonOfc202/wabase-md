# WABaseMD
Base para whatsapp multidevice em JS usando a lib [adiwajshing/baileys-md](https://github.com/adiwajshing/baileys/tree/multi-device)

## Instalação
```bash

#----------- CLONAR O REPOSITÓRIO -----------
git clone https://github.com/guiguicdd/WABaseMD.git
cd WABaseMD

#----------- INSTALAR AS DEPENDÊNCIAS -----------
yarn

#----------- INSTALAR TS NO REPO DO BAILEYS-MD -----------
cd node_modules/@adiwajshing/baileys-md
npm i typescript -g
tsc
cd ../../../

#----------- INICIAR O WABASEMD -----------
npm start

```

# ⚠ ATENÇÃO ⚠
